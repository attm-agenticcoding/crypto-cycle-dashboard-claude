# Claude Crypto Cycle Dashboard

A public, auto-updating dashboard tracking BTC / ETH cycle position: regime and
verdict, bottom / top scores (0–100), a reconciled bottom estimate with its
derivation, deployment guidance (as percentages), a historical DCA track record,
and supporting charts.

**Live:** https://attm-agenticcoding.github.io/crypto-cycle-dashboard-claude/

Only model outputs, market prices, and percentages are shown — no dollar amounts.

Refresh cadence:

- 06:30 ET live intraday snapshot;
- 09:30-16:00 ET live intraday snapshots every 30 minutes;
- 20:30 ET close snapshot, which writes the historical close record.

Research / educational only — not investment advice.
